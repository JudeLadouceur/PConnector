

In order to add a new character to the dropdowns, open the script in this folder called "Character Names". 
In this, just add a new character name anywhere in the script (with each one seperated by a comma, do not put a comma after the last one).
Doing this will add this new character to all character dropdowns.

In order to add a new variable to the dropdowns, open the script in this folder called "DialogueVaribles". 
In this, just add a new variable name anywhere in the script (with each one seperated by a comma, do not put a comma after the last one).
Doing this will add this new variable to all variable dropdowns.
